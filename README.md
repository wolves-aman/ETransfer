## 服务端（server目录）

### 安装
```shell
cd ./server && npm install
```
### 运行
```shell
node  ./server/server.js
```

## 客户端 （client目录）
Vue3 + PrimeVue

### 安装
```shell
cd ./client && yarn install
```

### 运行
```shell
cd ./client && yarn serve
```

### 编译
```shell
cd ./client && yarn build
```
